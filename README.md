# 🎮 AgentForge — Retro AI Multi-Agent App Builder

## 🚀 Quick Start

1. Get a free API key from https://aistudio.google.com/app/apikey
2. Open `.env` and paste it:
   ```
   GEMINI_API_KEY=your-key-here
   ```
3. Start the server:
   - **Windows:** double-click `start.bat`
   - **Mac/Linux:** run `node server.js` in terminal
4. Open `http://localhost:3000`

## 🔒 Security
- API key lives only in `.env` — never touches the browser
- Server blocks direct access to `.env` and `server.js`
- `.env` is gitignored so it won't be accidentally committed
